Opis szczegółowy

template<class T, int R>
class UkladRownanLiniowych< T, R >

Klasa realizuje pojęcie układu równań liniowych.

Układ równań liniowych opisywany jest przy użyciu kwadratowej macierzy współczynników wektora wyrazów wolnych oraz wyliczonego wektora rozwiązań.
